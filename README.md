# Homebrew Tap for Keyva CLI

This is the official Homebrew tap for [Keyva CLI](https://github.com/seracreativo/keyva-cli).

## Installation

```bash
brew tap seracreativo/keyva
brew install keyva
```

## Usage

```bash
keyva project list
keyva export --project "MyApp" --env "prod" --format env
```

See the [Keyva CLI documentation](https://github.com/seracreativo/keyva-cli) for more details.

## Requirements

- macOS 14.0 (Sonoma) or later
- Signed in to iCloud with the same Apple ID as the Keyva app
